Intro to CSS

CSS stands for Cascading Style Sheets. It is a stylesheet language that is used to describe the visual presentation of a web page written in HTML (Hyper Text Markup Language). HTML creates the structure of the page, while CSS adds styling to that structure.

Adding CSS to HTML

There are three ways to add CSS to HTML
  • Inline Styling - This is used in an HTML page using style attribute.
  • Internal Styling - It Means to add a style tag and place the HTML page's style within it.
  • External Styling - It means to create a .css file and place all styles inside the .css file.
